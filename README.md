# Pico-PLC-FX1N
 Orange pi Pico PLC Firmware
 Technical specifictions
  
  1.Hardware
   - Raspberry pi Pico

  2.PLC Function compatibility
   - PLC Model : FX1N/FX1NC
   - Input 10 Input
   - Output 7 Output
   - Ladder step : 2000 steps
   - Retentive memory : No
   - RS485 modbus : Yes
   - RS485 function
     - Modbus master
     - Modbus slave
   - Analog input 3 Input
